# Cthulhu Wars Solo — GitHub Pages build

Static build of the Library at Celaeno solo client, served from this branch.

Live: https://gjremus.github.io/cthulhu-wars/

To refresh after solo/ changes on `library-at-celaeno`:
1. `cd "Library at Celaeno/solo" && sbt fastLinkJS`
2. Re-run `scripts/publish-pages.sh` (or repeat the manual steps in the commit message).
